
# DigiTools

![DigiTools Banner](https://via.placeholder.com/800x200.png?text=DigiTools+Banner)

## A Little Description
DigiTools is a modern digital marketplace that helps creators, professionals, and businesses access premium AI-powered tools, design assets, templates, and productivity software—all in one place. Supercharge your workflow and create faster today!

---

## ⚙️ Technologies Used
- React.js  
- Tailwind CSS, DaisyUI 
- JavaScript (ES6+)  
- React-Toastify (NPM Package)  
- JSON (for product data)  


---

## Features

### 1. Premium Digital Tools
Access a curated collection of AI tools, templates, and productivity software designed to enhance creativity and efficiency.  

### 2. Interactive Shopping Cart
Add products to your cart, get real-time notifications, and checkout seamlessly. The cart tracks totals dynamically and allows item removal.  

### 3. Dynamic Stats and Active Users
Showcase live statistics such as active users, premium tools, and ratings with animated counters for a modern, engaging interface.  

---

## Components Overview
- **Navbar** – Navigation with cart indicator  
- **Banner** – Hero section with call-to-actions  
- **ActiveUsers** – Animated user, tools, and rating counters  
- **Products** – Product listing with toggleable cart view  
- **Pricing** – Subscription plans with featured highlights  
- **Workflow** – Steps or features presentation  
- **Footer** – Company info, product links, and social icons
